### Example

- Allow full access

```txt
User-agent: *
Disallow:
```

- Bloack all access

```txt
User-agent: *
Disallow: /
```

- Block one folder and Block one file

```txt
User-agent: *
Disallow: /folder/

User-agent: *
Disallow: /file.html
```
