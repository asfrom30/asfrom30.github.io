# Non-Reactive Data

```html
<template>
  <ul>
    <li v-for="item in $options.myArray">{{ item }}</li>
  </ul>
</template>
```

```js
export default {
  data() {
    return {
      someReactiveData: [1, 2, 3]
    };
  },
  //custom option name myArray
  myArray: null,
  created() {
    //access the custom option using $options
    this.$options.myArray = ["value 1", "value 2"];
  }
};
```
