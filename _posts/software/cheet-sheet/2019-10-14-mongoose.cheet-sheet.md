# Query

### Make Query

```js
const query = Device.find();
query
  .where("vrHmdId")
  .equals(vrHmdId)
  .where("ownerId")
  .equals(ownerId);

Device.findOneAndUpdate(query).then(dbRes => {
  console.log(dbRes);
});
```
