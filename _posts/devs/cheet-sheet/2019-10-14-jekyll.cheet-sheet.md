# Loop

### Category Loop

```liquid
{% raw %}

{% assign posts = site.posts | where_exp: "post", "post.categories contains 'Foo'" %}
{% for post in posts %}
  <li>{{ post.title }}</li>
{% endfor %}

{% endraw %}
```

# Date

### Date.now() to String

```liquid
{% raw %}
{{ site.time | date_to_string }}
{% endraw %}
// expected output : {{ site.time | date_to_string }}

{% raw %}
{{ page.date | date: '%B %d, %Y' }}
{% endraw %}
// expected output : {{ page.date | date: '%B %d, %Y' }}
```
