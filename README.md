# Introduction

## CLI

```
$ bundle install
$ bundle update // update dependency
$ bundle exec jekyll serve --livereload
```

## Structure

### Type of Navigation

- post
  - cf) welcome(welcome.page.html), portfolio, resume...
- router
  - cf) book (book.router.md)
  - see `router-provider` folder
- external link
  - cf) camping location
