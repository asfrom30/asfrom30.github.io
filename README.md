### Setup and Running

**Setup**

- install rvm
- require ruby 2.7
  - `$ rvm install 2.71`

**Install Dependency**

```sh
$ . bin/install.sh
```

**Build**

```sh
$ . bin/build.sh
```

**Livereload**

```sh
$ . bin/livereload.sh
```
